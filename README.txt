# KincoExampleYearoftheTiger
Animation und Sprachumschaltung

Dieses Beispielprogramm für ein Kinco MK070E zeigt, wie man leicht eine Sprachumschaltung und Bewegungsanimation (hier Laufschrift) erzeugen kann:
* Laufschrift in Form einer Bewegungsanimation (Bewegliche Komponente mit Beschriftung)
* Änderung der Position der Animation durch einen Timer (Funktion "periodisches Springen")
* Einfache Sprachumschaltung über Textbibliothek mit 3 Sprachen (Systemvariable LW9130 zum Umschalten)
* Multizustandschalter mit Anzeige (Multiple State Switch) als Dropdown nach oben öffnend mit Beschriftung zur Sprachauswahl
* Individuelles Bild 
* Zeitanzeige mit passender Formatierung

Das Projekt wurde für das Kinco HMI MK070E mit einer Auflösung von 800 x 480 Pixeln erstellt. Die Kinco HMI GL070E, G070E und GH070E haben eine gleiche Auflösung. Das Projekt kann für diese Modelle einfach per Rechtsklick auf das HMI in DTools konvertiert werden. 

Für HMI mit anderer Auflösung ist eine Konvertierung über diesen Weg ebenfalls möglich, es werden aber wahrscheinlich Nacharbeiten notwendig, um die Darstellung an das HMI anzupassen.

Das Programm wurde in der Entwicklungssoftware Kinco DTools 3.5.3 erstellt.
